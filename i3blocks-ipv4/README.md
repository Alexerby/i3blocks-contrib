# i3blocks-ipv4

Grabs IPv4 from ifconfig.

## Configuration
```ini
[ipv4]
label=IPv4
interval=once
command=$SCRIPT_DIR/i3blocks-contrib/i3blocks-ipv4/i3blocks-ipv4
```
